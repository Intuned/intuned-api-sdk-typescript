# Month

Months, one of JANUARY, FEBRUARY, MARCH, APRIL, MAY, JUNE, JULY, AUGUST, SEPTEMBER, OCTOBER, NOVEMBER, DECEMBER


## Supported Types

### `operations.Month1`

```typescript
const value: operations.Month1 = "FEBRUARY";
```

### `operations.Month2`

```typescript
const value: operations.Month2 = {
  start: "JUNE",
  step: 136825,
  end: "APRIL",
};
```

### `operations.Month3`

```typescript
const value: operations.Month3 = {
  start: "DECEMBER",
};
```

### `operations.Month4[]`

```typescript
const value: operations.Month4[] = [];
```

### `operations.Month5`

```typescript
const value: operations.Month5 = "*";
```
