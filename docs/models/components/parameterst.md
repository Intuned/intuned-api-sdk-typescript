# ParametersT

The parameters to be passed to the API.

## Example Usage

```typescript
import { ParametersT } from "@intuned/client/models/components";

let value: ParametersT = {
    key: "<value>",
};
```

## Supported Types

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = /* values here */
```

### `any[]`

```typescript
const value: any[] = /* values here */
```
